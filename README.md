`tokio-snappy`
==============

Wrap [rust-snappy](https://crates.io/crates/snap) for `tokio::io::AsyncRead` and `tokio::io::AsyncWrite`.

Read [rust-snappy](https://crates.io/crates/snap) and [snappy](https://github.com/google/snappy) for more.
